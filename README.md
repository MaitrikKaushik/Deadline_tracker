# Deadline Tracker

Keep all your deadlines in one place.

## Features

- **Add deadlines** — enter a title, optional description, due date, and priority (Low / Medium / High)
- **Urgency colour-coding** — cards are automatically colour-coded:
  - 🔴 Red border — overdue
  - 🟠 Orange border — due within 2 days
  - 🟡 Yellow border — due within 7 days
  - 🟢 Green border — due later
- **Complete / active toggle** — mark a deadline done and back again
- **Delete** — remove a deadline
- **Filter** — view All, Active, or Completed deadlines
- **Sort** — sort by date (ascending/descending) or by priority
- **Summary bar** — see counts of active, completed, and overdue items at a glance
- **Persistent** — deadlines are saved in `localStorage` and survive page reloads

## Usage

Open `index.html` in any modern browser — no build step or server required.
